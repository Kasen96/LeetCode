> 官方的dp解释的很不清晰。
> `dp[i]` 其实表示的是，以 `A[i]` 结尾的等差数列个数： 
> 考虑 `A=[3,4,5,6,7,8,9]`, 
> 当前已经计算出 `dp[2]=1`, `dp[3]=2`,需要求 `dp[4]`。
> `dp[4]=dp[3]+1` 的原因是： 
> 以 `A[3]=6` 结尾的等差数列已经有了 `dp[3]=2` 个：`[3,4,5,6]` 和 `[4,5,6]`。
> 想要知道以 `A[4]=7` 结尾的等差数列个数，那么需要在已经有的 `dp[3]` 个等差数列的尾部接上一个 `A[4]=7`, 
> 还有一个新等差数列 `[5,6,7]`，每次都会产生这个长度为 3 的等差数列。 所以才有了 `dp[i-1]+1`。
